## About

Cairo-json is a JSON library written in Cairo. It is useful for encoding and outputting JSON objects for Starknet such as Metadata.

## Getting Started

### 📦 Requirements

- [asdf-scarb](https://github.com/software-mansion/asdf-scarb)

### ⛏️ Compile

```bash
scarb build
```

## 🌡️ Test

```bash
scarb test
```
